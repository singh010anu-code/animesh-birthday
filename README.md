# 🎉 Animesh's Birthday Gift Registry

An interactive gift registry for Animesh's birthday celebration! Friends and family can browse gifts, view shopping links, and reserve what they'd like to give.

## Features

- ✨ Beautiful invitation popup with party details
- 🎁 15 curated gift items with direct shopping links
- 💙 Click to reserve gifts (blue button)
- 💛 Deep yellow button when gift is reserved
- 💾 Local storage saves reservations
- 📱 Fully responsive design
- 🎨 Modern gradient UI

## Tech Stack

- React 18
- Vite
- CSS3

## Getting Started

### Install Dependencies
```bash
npm install
```

### Run Dev Server
```bash
npm run dev
```

Visit `http://localhost:3000` to see the registry.

### Build for Production
```bash
npm run build
```

## Deploy on Replit

1. Create a Replit project
2. Import from GitHub: `https://github.com/singh010anu-code/animesh-birthday-registry`
3. Click "Run" - Replit will automatically install and start the server
4. Share the live URL with friends and family!

## How It Works

- Open the registry and see the birthday invitation
- Browse 15 gift options with prices and links
- Click "Reserve" to claim a gift (enter your name)
- Click "View Link" to shop for the gift
- Reserved gifts turn deep yellow and show who reserved them
- All reservations save to your browser's local storage

Enjoy the party! 🥳
